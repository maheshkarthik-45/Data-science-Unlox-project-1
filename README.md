# 📊 WhatsApp Chat Analyzer

A simple Python project to analyze exported WhatsApp chat files and generate useful chat statistics.

## Features
- Total message count
- Participant-wise messages
- Media message count
- Deleted message count
- Busiest day analysis
- Chat summary

## Technologies
- Python
- NumPy
- Jupyter Notebook

## How to Run
1. Clone this repository.
2. Place your exported WhatsApp chat (.txt) file in the project folder.
3. Open `unlox_project.ipynb`.
4. Run all cells.
